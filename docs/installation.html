Installation instructions

0. Install the Robot Operating System (ROS). Please choose the recommended "Desktop-Full Installation": Official installation instructions

1. We need to install some important packages. Open a terminal (ctrl+alt+T) and insert line-by-line. After each line press enter:

sudo add-apt-repository ppa:mystic-mirage/pycharm

sudo apt-get update

sudo apt-get install python-pip python-numpy pycharm-community python-scipy python-matplotlib libprotobuf-dev protobuf-compiler libboost-all-dev build-essential ros-kinetic-ivcon ros-kinetic-trac-ik* ros-kinetic-moveit* ros-kinetic-convex-decomposition ros-kinetic-pr2-controller-manager ros-kinetic-pr2-controllers-msgs ros-kinetic-pr2-controller-interface python-dev swig python-pygame git ros-kinetic-gazebo-ros-control ros-kinetic-controller-manager ros-kinetic-joint-state-controller

2. We need further packages for mathematic and scientific calculations. Please enter following line in the terminal and press enter:

sudo pip install numpy scipy matplotlib protobuf

3. Next, we install a simple simulator. Please enter line by line:

git clone https://github.com/pybox2d/pybox2d
cd pybox2d
python setup.py build
sudo python setup.py install
rosdep update
